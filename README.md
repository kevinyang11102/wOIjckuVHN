# 智慧社区系统简介 java364

## 项目概述

智慧社区系统是一个集成管理员与客户角色的社区管理平台，旨在通过高效的信息管理和优质的服务，提升社区管理水平和居民生活质量。

## 技术栈

- 后端：Spring Boot
- 前端：Vue 2 + ElementUI
- 运行环境：Node.js 14
- 数据库：MySQL 5.7以上
- JDK版本：1.8

## 功能模块

### 管理员角色

- **个人中心**：管理员账户信息管理与设置
- **客户管理**：管理客户信息
- **在线报修管理**：处理与跟踪报修请求
- **投诉建议管理**：处理客户投诉与建议
- **车位信息管理**：车位信息维护
- **车位租买管理**：租买请求处理
- **社区信息管理**：发布与通知管理
- **周边服务管理**：社区周边服务信息管理
- **问卷调查管理**：设计、发布和收集问卷
- **参与答卷管理**：整理和分析用户答卷
- **爱心助老管理**：组织爱心助老活动
- **留言板管理**：留言审查与回复
- **系统管理**：系统设置与权限分配

### 客户角色

- **车位信息**：查看车位信息
- **社区信息**：浏览社区通知
- **周边服务**：查看周边服务信息
- **问卷调查**：参与问卷调查
- **爱心助老**：了解与参与助老活动
- **通知公告**：查看社区通知和公告
- **留言反馈**：在留言板上留言
- **个人中心**：管理个人账户信息
- **在线报修管理**：提交与跟踪报修请求
- **投诉建议管理**：提交投诉和建议
- **车位租买管理**：租买车位
- **参与答卷管理**：参与问卷调查
- **我的收藏管理**：管理个人收藏
